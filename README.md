## Installation guide

Installation:

```bash
# clone repository
$ git clone https://github.com/JoeyQuandt/Frontend-Dev-Case-DeptAgency.git && cd front-dev-case

# install dependencies
$ npm install

$ npm run dev
```

Usage:

```bash
# development server
$ npm run dev
# build server
$ npm run build
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
